io.mesh_to_string(mesh: lagrange.core.SurfaceMesh, format: str = 'ply', binary: bool = True, exact_match: bool = True, selected_attributes: list[int] | None = None) → bytes#

Convert a mesh to a binary string based on specified format.

Parameters:

  • mesh – The input mesh.

  • format – Format to use. Supported formats are “obj”, “ply”, “gltf” and “msh”.

  • binary – Whether to save the mesh in binary format if supported. Defaults to True. Only msh, ply and glb support binary format.

  • exact_match – Whether to save attributes in their exact form. Some mesh formats may not support all the attribute types. If set to False, attributes will be converted to the closest supported attribute type. Defaults to True.

  • selected_attributes – A list of attribute ids to save. If not specified, all attributes will be saved. Defaults to None.

Return str:

The string representing the input mesh.

io.save_mesh(filename: os.PathLike, mesh: lagrange.core.SurfaceMesh, binary: bool = True, exact_match: bool = True, selected_attributes: list[int] | None = None) → None#

Save mesh to file.

Filename extension determines the file format. Supported formats are: obj, ply, msh, glb and gltf.

Parameters:

  • filename – The output file name.

  • mesh – The input mesh.

  • binary – Whether to save the mesh in binary format if supported. Defaults to True. Only msh, ply and glb support binary format.

  • exact_match – Whether to save attributes in their exact form. Some mesh formats may not support all the attribute types. If set to False, attributes will be converted to the closest supported attribute type. Defaults to True.

  • selected_attributes – A list of attribute ids to save. If not specified, all attributes will be saved. Defaults to None.

io.string_to_mesh(data: bytes, triangulate: bool = False) → lagrange.core.SurfaceMesh#

Convert a binary string to a mesh.

The binary string should use one of the supported formats. Supported formats include obj, ply, gltf, glb and msh. Format is automatically detected.

Parameters:

  • data – A binary string representing the mesh data in a supported format.

  • triangulate – Whether to triangulate the mesh if it is not already triangulated. Defaults to False.

Return SurfaceMesh:

The mesh object extracted from the input string.
